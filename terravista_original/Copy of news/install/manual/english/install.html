Installing NukeAddOn

Warning!

For this AddOn you will run a NukeAddOn installer which is still in BETA status. Please do some backup toward your database before you perform the upgrade.

Base Requirements

In order to setup Nuke Add-On the folowing prerequisits are necessary.

PHP Nuke 4.4.1a PHPNuke.ORG database only.
PHP Nuke 5.0 PHPNuke.ORG files in your server.

We are sorry but due to the nature of PHP Nuke 5.0 the NukeAddOn 5.0 is not compatible with any previous PHP Nuke.

Installing the Package

Unzip the package (addon503.zip) into the directory you want to use on you web server, if you're unsure where that is, ask you system administrator.

Replace the old .php files with the new one!

You must have PHP Nuke 5.0 install or 4.4.1a database only:

1) run from your browser http://www.yourdomain.com/installer.php
2) Follow 5 simple step instruction
3) If all when well NukeAddOn should run perfectly
4) Please after the installation delete the installer.php and directory install for security reason.

Please also SIDEBOX and WEATHER if you need to get more information on using the sidebox and weather.
