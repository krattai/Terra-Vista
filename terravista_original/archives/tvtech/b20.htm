VRML 2.0 Browsers

  • CosmoPlayer - Silicon Graphics, Inc.
    Hurry up and get your ship mods from Fleet Command, Cosmonauts! Windows NT and Win95.
  • CyberPassage - Sony
    Sony has been kind enough to include a VRML 1.0 to VRML 2.0 converter. CyberPassage is available for Windows NT and Win95.
  • Liquid Reality -Dimension X
    Rather than a VRML 2.0 browser, Liquid Reality is a set of Java class libraries for creating programs that support VRML 2.0. It is available for Win95/NT, Solaris, and Linux, with a Mac version in the works.

      CosmoPlayer and CyberPassage are compliant with Draft #2 of the VRML 2.0 spec as released 5/30/96. Liquid Reality supports a majority of the VRML 2.0 nodes.
