Planting yourself in Cyberspace



Congratulations on acquiring your Zone. Now that the files have been extracted the next thing to do will be to stake your claim in CyberSpace. You will need the following in order to plant your Zone:

+ the VRML (*.wrl) files which came with these instructions
+ Web space on your local internet provider or other host
+ an FTP client to upload the files

The following are step by step instructions. If you know the process to upload the files to the server, you can skip over this section.

+ ensure your FTP client software is installed and set up
++ this information should have come along with your FTP software
+ log in to your Internet provider and start your FTP software
+ create a directory for the VRML files on your host space
+ change directories on your local computer to the directory you installed these files to
+ change directories on your host server to the directory you wish to upload these files to
+ go ahead and upload the files

Congratulations, you have now staked your claim in CyberSpace.

At this point, you will want to view the Zone to ensure the Zone has successfully uploaded and is being served properly by your host. To do this, start up your browser software and log on to your server as normal. Now, in the address location, type in your host URL with the path to the files, and the name of the base wrl. The base wrl will depend on which Zone you have chosen. So the address will look something like this:
http://terravista.dhs.org/user/me/VRML/gtbasic.wrl

If everything is working correctly, congratulations! If things do not seem quite right, please contact the email address at the bottom of this page for further help and instructions.

Once you have the files uploaded, you will want to get them linked to other Zones. For this, you will need to contact us at Terra Vista. Please email:
