Weather Information

In order for you to change the Country of your choice you need to edit your config.php files and change your $accid = "WIII"; with your country code which can be found at MSNBC Weather at MSNBC Weather click on "Find Local Weather" to find your weather location and the accid id will appear to you in the browser box.

"WIII" is accid code for Jakarta, Indonesia.
"LAX" is accid code for Los Angeles Coast.
"59L" is accid code for Los Angeles Downtown.
"SFO" is accid code for San Francisco.
"WSAP" is accid code for Singapore.

Enjoy!
