Forum Configuration


With Forum Configuration you can set your forum configuration.

Just edit the list of configuration you want and it should change your forum configuration. For forum themes you must edit your theme.php color setting manually.

This are some of detail information:
Allow HTML: which allow user to post HTML in the forum post and replies.
Allow BBCODE: which allow user to post BBCODE in the forum post and replies.
Allow Signature: which allow to user to include their signature in the forum post and replies.
Hot Topic Threshold: the minimum number of replies before its topic become HOT topic.
Posts per Page: the number of posts per topic that will be displayed per page of a topic.
Topics per Forum: the number of topic list per pages.
